## Supported Messengers

| Messenger | Status |
|-----------|--------|
| **Telegram** | Available |
| **Viber** | Available |
| **WhatsApp** | Coming soon |

## Telegram Setup

1. Create a bot via [@BotFather](https://t.me/BotFather)
2. Copy the bot token
3. Go to **Integrations** → **Add Integration** → **Telegram**
4. Paste your bot token
5. Assign an AI agent

The webhook is configured automatically. Your bot will start responding to messages immediately.

## Viber Setup

1. Create a bot in the [Viber Admin Panel](https://partners.viber.com/)
2. Copy the authentication token
3. Go to **Integrations** → **Add Integration** → **Viber**
4. Paste your token
5. Assign an AI agent

## Message Types

AI agents can send:
- Text messages
- Images and documents (from knowledge base)
- Quick reply buttons
