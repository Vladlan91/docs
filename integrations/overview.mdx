## Overview

Revol integrations allow your AI agents to communicate through multiple channels beyond the website widget.

## How It Works

1. **Connect** — Add credentials for your provider
2. **Configure** — Map channels to AI agents
3. **Receive** — Incoming messages/calls are routed to the assigned agent
4. **Respond** — Agent processes and responds through the same channel

## Webhook Security

All webhook endpoints are protected with:
- **Signature validation** — Verify request authenticity
- **Idempotency** — Prevent duplicate processing
- **Rate limiting** — 120 requests per minute per channel

<Note>
  Integrations require a **Premium** plan or higher.
</Note>
