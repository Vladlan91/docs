## Visual Settings

Customize your widget in **Admin Panel** → **General Settings** → **Widget Customization**.

### Colors

| Setting | Description |
|---------|-------------|
| **Button Color** | Primary action button color |
| **Animation Color** | Pulse and glow effects |
| **Background Color** | Chat window background |
| **Message Background** | Assistant message bubble color |

### Layout

| Setting | Description |
|---------|-------------|
| **Border Radius** | Widget corner roundness (0-20px) |
| **Position** | Bottom-left or bottom-right |
| **Widget Language** | Interface language (EN, UK, PL, DE, ES) |

### Branding

| Setting | Description |
|---------|-------------|
| **Avatar** | Agent avatar image |
| **Welcome Message** | Initial greeting text |
| **Branding** | "Powered by Revol" badge (removable on Premium+) |

## Dark Mode

The widget automatically detects the user's system preference and switches between light and dark themes.
