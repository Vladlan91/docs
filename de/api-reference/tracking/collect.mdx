---
title: Ereignisse erfassen
api: "POST /api/tracker/collect"
description: "Tracking-Daten senden (Seitenaufrufe, Ereignisse)"
---

## Anfrage

<ParamField body="api_key" type="string" required>
  Ihr Widget API-Schluessel
</ParamField>

<ParamField body="company_id" type="integer" required>
  Ihre Unternehmens-ID
</ParamField>

<ParamField body="session_token" type="string" required>
  Eindeutiger Sitzungsbezeichner
</ParamField>

<ParamField body="pageviews" type="array">
  Array von Seitenaufruf-Objekten (max. 50)
</ParamField>

<ParamField body="events" type="array">
  Array von Ereignis-Objekten (max. 50)
</ParamField>

## Seitenaufruf-Objekt

```json
{
  "url": "https://example.com/products",
  "title": "Our Products",
  "referrer": "https://google.com",
  "timestamp": 1709640000
}
```

## Ereignis-Objekt

```json
{
  "name": "form_submit",
  "category": "engagement",
  "data": {
    "form_id": "contact-form",
    "fields": ["name", "email", "phone"]
  },
  "timestamp": 1709640000
}
```

## Antwort

```json
{
  "success": true
}
```

<Warning>
  Arrays sind auf jeweils 50 Eintraege begrenzt, um Missbrauch zu verhindern.
</Warning>
