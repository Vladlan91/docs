---
title: Nachricht senden
api: "POST /api/widget/send-message"
description: "Eine Nachricht an den KI-Agenten senden"
---

## Anfrage

<ParamField body="api_key" type="string" required>
  Ihr Widget API-Schluessel
</ParamField>

<ParamField body="company_id" type="integer" required>
  Ihre Unternehmens-ID
</ParamField>

<ParamField body="message" type="string" required>
  Nachrichtentext des Benutzers (max. 2000 Zeichen)
</ParamField>

<ParamField body="session_id" type="string">
  Sitzungsbezeichner fuer die Gespraechskontinuitaet
</ParamField>

## Antwort

Die Antwort wird als Server-Sent Events (SSE) gestreamt:

```
data: {"chunk": "Hello"}
data: {"chunk": "! How"}
data: {"chunk": " can I"}
data: {"chunk": " help you?"}
data: [DONE]
```

## Beispiel

```javascript
const response = await fetch('/api/widget/send-message', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    api_key: 'YOUR_API_KEY',
    company_id: 1,
    message: 'What products do you offer?',
    session_id: 'abc123'
  })
});

const reader = response.body.getReader();
const decoder = new TextDecoder();

while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  console.log(decoder.decode(value));
}
```
