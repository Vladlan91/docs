## Widget API-Schluessel

Die Widget API verwendet einen unternehmensspezifischen API-Schluessel zur Authentifizierung. Dieser Schluessel wird vom Tracker-Skript und Widget verwendet.

```html
<script src="https://app.revol.com/tracker.js"
        data-api-key="YOUR_API_KEY"
        data-company-id="YOUR_COMPANY_ID"
        async>
</script>
```

### Domain-Validierung

Widget API-Anfragen werden anhand des **Origin**-Headers validiert. Nur Anfragen von Domains, die in Ihren Unternehmenseinstellungen registriert sind, werden akzeptiert.

## API Endpoints

Widget API endpoints akzeptieren den API-Schluessel ueber:

```bash
# Query parameter
GET /api/widget/init?api_key=YOUR_API_KEY&company_id=YOUR_COMPANY_ID

# Oder im Anfrage-Body
POST /api/widget/send-message
Content-Type: application/json

{
  "api_key": "YOUR_API_KEY",
  "company_id": "YOUR_COMPANY_ID",
  "message": "Hello"
}
```
