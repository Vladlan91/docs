## Basis-URL

```
https://app.revol.com/api
```

## Authentifizierung

Alle API-Anfragen erfordern eine Authentifizierung ueber einen API-Schluessel, der im Anfrage-Header oder als query parameter uebergeben wird.

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
     https://app.revol.com/api/widget/init
```

## Ratenbegrenzungen

| Endpoint | Limit |
|----------|-------|
| Widget API | 15 Anfragen/Min |
| Tracking API | 60 Anfragen/Min |
| Allgemeine API | Abhaengig vom Tarif |

## Antwortformat

Alle Antworten sind JSON:

```json
{
  "success": true,
  "data": { ... }
}
```

Fehlerantworten:

```json
{
  "success": false,
  "error": "Error message"
}
```
