## Overview

The Sessions tab provides a detailed view of all visitor sessions. Browse, filter, and drill down into individual visitor journeys to understand how people interact with your website.

## Sessions Table

Each row represents a single visitor session:

| Column | Description |
|---|---|
| **Guest Token** | Unique visitor identifier |
| **Country** | Visitor's country (flag icon) |
| **Device** | Desktop, Mobile, or Tablet |
| **Browser** | Browser name and version |
| **OS** | Operating system |
| **Pages** | Number of pages viewed |
| **Duration** | Session length |
| **Source** | Traffic source (UTM or referrer) |
| **Started** | Session start time |

## Session Detail

Click on any session to open the detail view showing the complete visitor journey:

- **Pageviews** — ordered list of pages visited with timestamps and time spent on each page
- **Events** — custom events triggered during the session
- **Scroll depth** — how far the visitor scrolled on each page
- **Conversation** — link to the chat conversation if one was started

## Filters

Filter sessions by device, country, source, browser, OS, bounce status, campaign, and date range — the same filters available on the Dashboard.
