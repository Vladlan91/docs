## Overview

The Revol Tracker monitors visitor activity on your website. It collects session data, pageviews, and custom events — all through the same script that powers the widget.

## What Gets Tracked

### Sessions
- Device type, browser, OS
- Geolocation (country, city)
- UTM parameters (source, medium, campaign)
- Referrer URL

### Pageviews
- Page URL and title
- Time on page
- Scroll depth

### Events
- Form submissions
- Button clicks
- Phone number reveals
- Custom events

## Campaigns

Sessions are automatically grouped into **campaigns** based on UTM parameters:

| Parameter | Example |
|-----------|---------|
| `utm_source` | google, facebook |
| `utm_medium` | cpc, email |
| `utm_campaign` | spring_sale |
| `utm_term` | ai chatbot |
| `utm_content` | banner_v2 |

## Triggers

Automated actions based on visitor behavior:

- **Time on page** — Show overlay after N seconds
- **Scroll depth** — Trigger when user scrolls past threshold
- **Exit intent** — Detect when user is about to leave
- **Page count** — Trigger after N pageviews
