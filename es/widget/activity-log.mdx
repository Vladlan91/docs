## Overview

The Activity Log tab shows a real-time feed of the last 500 visitor events on your website. Use it to monitor live activity, debug event tracking, or verify that triggers and events are firing correctly.

  <img className="hidden dark:block" src="/images/tracker-activity-log-dark.png" alt="Activity log" />


## Log Entries

Each entry in the activity log shows:

| Column | Description |
|---|---|
| **Time** | Timestamp of the event |
| **Type** | Event type: Pageview, Event, Trigger, Conversation, Phone Swap |
| **Details** | Event-specific information (page URL, event name, trigger name, etc.) |
| **Session** | Link to the associated session |
| **Device** | Visitor's device type |
| **Country** | Visitor's country |

## Filters

Filter the activity log by event type to focus on specific activity:

- **All** — show all events
- **Pageviews** — page loads only
- **Events** — custom events only
- **Triggers** — trigger activations only
- **Conversations** — chat conversations started
- **Phone Swaps** — phone number substitutions

## Usage

The Activity Log is useful for:

- **Debugging** — verify that events and triggers fire as expected
- **Live monitoring** — watch visitor activity in real time
- **Quick audits** — check recent activity without navigating through sessions
