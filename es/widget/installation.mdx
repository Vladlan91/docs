## Instalación rápida

Agregue este único script antes de la etiqueta de cierre `</head>` en su sitio web:

```html
<script src="https://app.revol.com/tracker.js"
        data-api-key="YOUR_API_KEY"
        data-company-id="YOUR_COMPANY_ID"
        async>
</script>
```

Este script gestiona tanto el **widget de chat** como el **seguimiento de visitantes** — no se requiere código adicional.

## Configuración

| Atributo | Obligatorio | Descripción |
|-----------|----------|-------------|
| `data-api-key` | Sí | Su clave API del widget |
| `data-company-id` | Sí | Su ID de empresa |

<Tip>
  Encuentre su clave API e ID de empresa en **Panel de administración** → **Configuración general**.
</Tip>

## Qué se carga

El script de seguimiento automáticamente:

1. **Inicializa el seguimiento** — sesiones, vistas de página, eventos
2. **Carga el widget** — burbuja de chat con su apariencia personalizada
3. **Carga módulos bajo demanda** — triggers, overlays, intercambio de teléfono (solo cuando se necesitan)

## Shadow DOM

El widget se renderiza dentro de un contenedor **Shadow DOM**, lo que significa:

- El CSS de su sitio no afectará al widget
- El CSS del widget no afectará a su sitio
- Aislamiento completo de estilos

## Aplicaciones de página única (SPA)

El rastreador detecta automáticamente los cambios de URL mediante eventos `pushState` y `popstate`. No se necesita configuración adicional para aplicaciones React, Vue o Angular.
