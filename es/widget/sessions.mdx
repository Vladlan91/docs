## Descripción General

La página de Sesiones proporciona una vista detallada de todas las sesiones de visitantes dentro de una campaña. Haga clic en cualquier fila de campaña en la pestaña Campañas para abrir un **visor de sesiones en pantalla completa** — una herramienta poderosa para entender cómo los visitantes navegan por su sitio web, qué páginas ven y si inician conversaciones con su agente de IA.

## Visor de Sesiones de Campaña

El visor de sesiones se abre como un modal de pantalla completa con el nombre de la campaña, la insignia de estado y el ID en el encabezado. El encabezado también proporciona acciones rápidas:

| Acción | Descripción |
|---|---|
| **Insertar** | Obtener el código de inserción para el enlace de seguimiento de esta campaña |
| **Vista Previa** | Previsualizar la página de destino de la campaña |
| **Configuración** | Acceso rápido a fuente de campaña, medio, ID y acciones de editar/eliminar |

El visor tiene dos pestañas principales: **En Vivo** y **Sesiones**.

---

## Pestaña En Vivo

La pestaña En Vivo muestra la **actividad de visitantes en tiempo real** para la campaña. Cuando los visitantes están navegando activamente su sitio web a través del enlace de seguimiento de esta campaña, sus transiciones de página aparecen aquí a medida que ocurren.

### Canvas En Vivo

El lado izquierdo muestra un gráfico de flujo de páginas en tiempo real. A medida que los visitantes navegan entre páginas, el gráfico se actualiza con conexiones animadas que muestran el camino de una página a otra.

**Controles del canvas:**

| Botón | Descripción |
|---|---|
| **Actualizar** | Recargar los datos en vivo |
| **Acercar / Alejar** | Ajustar el nivel de zoom del gráfico |
| **Restablecer Zoom** | Volver al zoom predeterminado |

Cuando no hay visitantes activos actualmente, el canvas muestra un estado vacío: *"No hay sesiones en vivo — Cuando los visitantes naveguen esta campaña, sus transiciones de página aparecerán aquí en tiempo real."*

### Panel de Visitantes Activos

El lado derecho muestra una lista de **visitantes activos** con un indicador en vivo (punto verde pulsante) y conteo. Cada tarjeta de visitante activo muestra su tipo de dispositivo, navegador y página actual.

## Pestaña Sesiones

La pestaña Sesiones muestra **todas las sesiones históricas** atribuidas a esta campaña. Aquí es donde analiza el comportamiento pasado de los visitantes en detalle.

### Gráfico de Flujo de Páginas

El lado izquierdo muestra un **gráfico visual de flujo de páginas** para la sesión seleccionada — una cadena vertical de nodos de página que muestra el recorrido del visitante por su sitio web.

#### Píldoras de Filtro

En la parte superior del gráfico, dos filas de píldoras de filtro interactivas le ayudan a acotar las sesiones:

| Filtro | Descripción |
|---|---|
| **Dispositivos** | Filtrar por Escritorio, Tableta o Móvil. Haga clic en una píldora para mostrar solo sesiones de ese tipo de dispositivo. |
| **Fuentes de Tráfico** | Filtrar por fuente — Google, Facebook, Instagram, TikTok, Telegram, Directo, y más de 20 otras fuentes. Cada fuente muestra su logotipo para identificación rápida. |

Los filtros activos se resaltan. Haga clic en una píldora para alternarla. La lista de sesiones y el gráfico se actualizan para mostrar sesiones coincidentes.

#### Nodos de Página

Cada página en el recorrido del visitante se muestra como un **nodo de página** con:

| Elemento | Descripción |
|---|---|
| **Número** | Número secuencial de página en la visita (1, 2, 3...) |
| **Título de Página** | El título HTML de la página |
| **URL de Página** | La ruta URL (por ejemplo, `/portfolio`, `/terrazzo`) |
| **Insignia de Tiempo** | Tiempo pasado en esta página (por ejemplo, "54s", "1m 47s") — mostrado a la izquierda |
| **Insignia de Scroll** | Porcentaje de profundidad de scroll (por ejemplo, "72%", "60%") — mostrado a la derecha |

Los nodos están conectados por **líneas conectoras SVG** que muestran el flujo de navegación. Haga clic en cualquier nodo de página para ver sus detalles completos en el panel derecho.

### Panel Derecho

El panel derecho tiene tres sub-pestañas: **Sesiones**, **Detalles** y **Conversaciones**.

#### Sub-pestaña Sesiones

Una lista desplazable de todas las sesiones de la campaña. Cada tarjeta de sesión muestra:

| Elemento | Descripción |
|---|---|
| **Icono de Fuente** | Logotipo de fuente de tráfico (Google, Facebook, Telegram, TikTok, Directo, etc.) |
| **Icono de Dispositivo** | Icono de escritorio, tableta o móvil |
| **Token del Visitante** | Identificador anónimo único del visitante (por ejemplo, `rvl_Bf0G`) |
| **Insignia de Rebote** | Etiqueta roja "Rebote" si el visitante vio solo una página |
| **Tiempo Transcurrido** | Tiempo relativo desde la sesión (por ejemplo, "hace 2d", "hace 15d") |
| **Conteo de Páginas** | Número de páginas vistas |
| **Duración** | Duración total de la sesión (por ejemplo, "6m 54s") |
| **Navegador** | Nombre del navegador (Chrome, Safari, Firefox, Edge, Samsung Internet) |
| **Ubicación** | Código de país y ciudad (por ejemplo, "UA, Kyiv", "DE, Hamburg") |

Haga clic en una tarjeta de sesión para cargar su gráfico de flujo de páginas a la izquierda. La sesión activa se resalta.

El encabezado muestra el conteo total de sesiones y un botón **Actualizar** para recargar la lista.

#### Sub-pestaña Detalles

Cuando hace clic en un **nodo de página** en el gráfico, la sub-pestaña Detalles muestra información completa sobre esa página vista específica:

**Información de Página:**

| Campo | Descripción |
|---|---|
| **Título** | Título completo de la página |
| **URL** | URL completa de la página (con enlace externo para abrir en nueva pestaña) |
| **Referencia** | La URL de la página anterior de donde vino el visitante |

**Engagement:**

| Campo | Descripción |
|---|---|
| **Tiempo en la página** | Cuánto tiempo permaneció el visitante en esta página |
| **Profundidad de scroll** | Qué tan abajo se desplazó el visitante (porcentaje) |
| **Visitado a las** | Fecha y hora exacta de la vista de página |

Si no se selecciona ningún nodo de página, el panel muestra: *"Seleccione un nodo de página para ver los detalles."*

#### Sub-pestaña Conversaciones

Si el visitante inició una **conversación de chat** con su agente de IA durante la sesión, la transcripción completa de la conversación aparece aquí — incluyendo tanto los mensajes del visitante como las respuestas del agente.

Si la sesión no tiene una conversación vinculada, el panel muestra: *"No hay conversación vinculada a esta sesión."*

<Tip>
  La sub-pestaña Conversaciones le permite ver el contexto exacto de una conversación — qué páginas navegó el visitante antes de iniciar el chat, cuánto tiempo pasó y qué discutió con el agente de IA. Esto es invaluable para entender la intención del visitante y mejorar las respuestas de su agente.
</Tip>

## Indicadores de Sesión

Las sesiones utilizan indicadores visuales para comunicar rápidamente información clave:

| Indicador | Significado |
|---|---|
| **Logotipo de fuente** | De dónde vino el visitante (Google, Facebook, Instagram, TikTok, Telegram, Directo, etc.) |
| **Icono de dispositivo** | Monitor de escritorio, tableta o teléfono móvil |
| **Insignia "Rebote"** | El visitante vio solo 1 página y se fue |
| **Visitante recurrente** | El mismo token `rvl_` apareciendo en múltiples sesiones |
| **Insignias de tiempo en nodos** | Tiempo pasado en cada página individual |
| **Insignias de scroll en nodos** | Qué tan lejos se desplazó el visitante en cada página |
