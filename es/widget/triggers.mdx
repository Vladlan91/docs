## Overview

The Triggers tab lets you create automated actions that fire based on visitor behavior rules. Triggers can display popups, overlays, notification bars, or custom HTML when specific conditions are met.

## Creating a Trigger

Click **Create Trigger** to set up a new automated action:

| Field | Description |
|---|---|
| **Name** | Trigger name for identification |
| **Type** | Display type: Popup, Overlay, Notification Bar, or Custom HTML |
| **Content** | The message or HTML content to display |

## Trigger Conditions

Define when the trigger should fire using one or more conditions:

| Condition | Description |
|---|---|
| **Time on Page** | Fire after the visitor spends X seconds on the page |
| **Scroll Depth** | Fire when the visitor scrolls past a percentage threshold |
| **Exit Intent** | Fire when the visitor moves the mouse toward the browser close/back area |
| **Page Count** | Fire after the visitor views X pages in the session |
| **Page URL** | Fire only on specific pages |
| **Device** | Fire only on Desktop, Mobile, or Tablet |
| **New/Returning** | Fire only for new or returning visitors |

## Trigger Settings

| Setting | Description |
|---|---|
| **Frequency** | How often to show: Once per session, Once per visitor, or Every time |
| **Delay** | Additional delay in seconds before displaying |
| **Enabled** | Toggle to activate or deactivate the trigger |

## Triggers List

The triggers table shows all defined triggers with:

| Column | Description |
|---|---|
| **Name** | Trigger name |
| **Type** | Display type |
| **Conditions** | Summary of trigger conditions |
| **Fires** | Total activations |
| **Enabled** | Active status |
| **Actions** | Edit or delete the trigger |
