---
title: Enviar mensaje
api: "POST /api/widget/send-message"
description: "Enviar un mensaje al agente de IA"
---

## Solicitud

<ParamField body="api_key" type="string" required>
  Su clave de API del widget
</ParamField>

<ParamField body="company_id" type="integer" required>
  Su ID de empresa
</ParamField>

<ParamField body="message" type="string" required>
  Texto del mensaje del usuario (maximo 2000 caracteres)
</ParamField>

<ParamField body="session_id" type="string">
  Identificador de sesion para la continuidad de la conversacion
</ParamField>

## Respuesta

La respuesta se transmite como Server-Sent Events (SSE):

```
data: {"chunk": "Hello"}
data: {"chunk": "! How"}
data: {"chunk": " can I"}
data: {"chunk": " help you?"}
data: [DONE]
```

## Ejemplo

```javascript
const response = await fetch('/api/widget/send-message', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    api_key: 'YOUR_API_KEY',
    company_id: 1,
    message: 'What products do you offer?',
    session_id: 'abc123'
  })
});

const reader = response.body.getReader();
const decoder = new TextDecoder();

while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  console.log(decoder.decode(value));
}
```
