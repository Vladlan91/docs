## Clave de API del Widget

La API del Widget utiliza una clave de API especifica de la empresa para la autenticacion. Esta clave es utilizada por el script de seguimiento y el widget.

```html
<script src="https://revol-ai.app/tracker.js"
        data-api-key="YOUR_API_KEY"
        data-company-id="YOUR_COMPANY_ID"
        async>
</script>
```

### Validacion de dominio

Las solicitudes a la API del Widget se validan contra el header **Origin**. Solo se aceptaran solicitudes provenientes de dominios registrados en la configuracion de su empresa.

## Endpoints de la API

Los endpoints de la API del Widget aceptan la clave de API mediante:

```bash
# Query parameter
GET /api/widget/init?api_key=YOUR_API_KEY&company_id=YOUR_COMPANY_ID

# O en el cuerpo de la solicitud
POST /api/widget/send-message
Content-Type: application/json

{
  "api_key": "YOUR_API_KEY",
  "company_id": "YOUR_COMPANY_ID",
  "message": "Hello"
}
```
