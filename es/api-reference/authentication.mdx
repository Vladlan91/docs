## Widget API Key

The Widget API uses a company-specific API key for authentication. This key is used by the tracker script and widget.

```html
<script src="https://app.revol.com/tracker.js"
        data-api-key="YOUR_API_KEY"
        data-company-id="YOUR_COMPANY_ID"
        async>
</script>
```

### Domain Validation

Widget API requests are validated against the **Origin** header. Only requests from domains registered in your company settings will be accepted.

## API Endpoints

Widget API endpoints accept the API key via:

```bash
# Query parameter
GET /api/widget/init?api_key=YOUR_API_KEY&company_id=YOUR_COMPANY_ID

# Or in request body
POST /api/widget/send-message
Content-Type: application/json

{
  "api_key": "YOUR_API_KEY",
  "company_id": "YOUR_COMPANY_ID",
  "message": "Hello"
}
```
