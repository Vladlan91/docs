---
title: Sesiones
api: "POST /api/tracker/session"
description: "Crear o actualizar una sesion de seguimiento"
---

## Solicitud

<ParamField body="api_key" type="string" required>
  Su clave de API del widget
</ParamField>

<ParamField body="company_id" type="integer" required>
  Su ID de empresa
</ParamField>

<ParamField body="session_token" type="string" required>
  Identificador unico de sesion (generado por tracker.js)
</ParamField>

<ParamField body="device" type="object">
  Informacion del dispositivo
</ParamField>

<ParamField body="utm" type="object">
  Parametros de campana UTM
</ParamField>

## Objeto de dispositivo

```json
{
  "type": "desktop",
  "browser": "Chrome",
  "os": "Windows 11",
  "screen": "1920x1080"
}
```

## Objeto UTM

```json
{
  "source": "google",
  "medium": "cpc",
  "campaign": "spring_sale",
  "term": "ai chatbot",
  "content": "banner_v2"
}
```

## Respuesta

```json
{
  "success": true,
  "data": {
    "session_id": 12345
  }
}
```
