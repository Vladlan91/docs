---
title: Recopilar eventos
api: "POST /api/tracker/collect"
description: "Enviar datos de seguimiento (vistas de pagina, eventos)"
---

## Solicitud

<ParamField body="api_key" type="string" required>
  Su clave de API del widget
</ParamField>

<ParamField body="company_id" type="integer" required>
  Su ID de empresa
</ParamField>

<ParamField body="session_token" type="string" required>
  Identificador unico de sesion
</ParamField>

<ParamField body="pageviews" type="array">
  Arreglo de objetos de vista de pagina (maximo 50)
</ParamField>

<ParamField body="events" type="array">
  Arreglo de objetos de evento (maximo 50)
</ParamField>

## Objeto de vista de pagina

```json
{
  "url": "https://example.com/products",
  "title": "Our Products",
  "referrer": "https://google.com",
  "timestamp": 1709640000
}
```

## Objeto de evento

```json
{
  "name": "form_submit",
  "category": "engagement",
  "data": {
    "form_id": "contact-form",
    "fields": ["name", "email", "phone"]
  },
  "timestamp": 1709640000
}
```

## Respuesta

```json
{
  "success": true
}
```

<Warning>
  Los arreglos estan limitados a 50 elementos cada uno para prevenir abusos.
</Warning>
