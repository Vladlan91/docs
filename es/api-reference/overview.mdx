## URL base

```
https://app.revol.com/api
```

## Autenticacion

Todas las solicitudes a la API requieren autenticacion mediante una clave de API enviada en el encabezado de la solicitud o como query parameter.

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
     https://app.revol.com/api/widget/init
```

## Limites de frecuencia

| Endpoint | Limite |
|----------|--------|
| Widget API | 15 solicitudes/min |
| Tracking API | 60 solicitudes/min |
| API general | Segun el plan |

## Formato de respuesta

Todas las respuestas son JSON:

```json
{
  "success": true,
  "data": { ... }
}
```

Respuestas de error:

```json
{
  "success": false,
  "error": "Error message"
}
```
