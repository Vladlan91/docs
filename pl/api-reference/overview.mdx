## Bazowy URL

```
https://app.revol.com/api
```

## Uwierzytelnianie

Wszystkie żądania API wymagają uwierzytelnienia za pomocą klucza API przekazanego w nagłówku żądania lub jako query parameter.

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
     https://app.revol.com/api/widget/init
```

## Limity żądań

| Endpoint | Limit |
|----------|-------|
| Widget API | 15 żądań/min |
| Tracking API | 60 żądań/min |
| Ogólne API | W zależności od planu |

## Format odpowiedzi

Wszystkie odpowiedzi są w formacie JSON:

```json
{
  "success": true,
  "data": { ... }
}
```

Odpowiedzi błędów:

```json
{
  "success": false,
  "error": "Error message"
}
```
