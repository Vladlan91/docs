## Klucz Widget API

Widget API używa klucza API przypisanego do firmy w celu uwierzytelnienia. Ten klucz jest używany przez skrypt trackera i widget.

```html
<script src="https://app.revol.com/tracker.js"
        data-api-key="YOUR_API_KEY"
        data-company-id="YOUR_COMPANY_ID"
        async>
</script>
```

### Walidacja domeny

Żądania Widget API są weryfikowane na podstawie nagłówka **Origin**. Akceptowane będą wyłącznie żądania z domen zarejestrowanych w ustawieniach Twojej firmy.

## Endpointy API

Endpointy Widget API akceptują klucz API poprzez:

```bash
# Query parameter
GET /api/widget/init?api_key=YOUR_API_KEY&company_id=YOUR_COMPANY_ID

# Lub w treści żądania
POST /api/widget/send-message
Content-Type: application/json

{
  "api_key": "YOUR_API_KEY",
  "company_id": "YOUR_COMPANY_ID",
  "message": "Hello"
}
```
