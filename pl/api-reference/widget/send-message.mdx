title: Wyślij wiadomość
api: "POST /api/widget/send-message"
description: "Wyślij wiadomość do agenta AI"
---

## Żądanie

<ParamField body="api_key" type="string" required>
  Twój klucz Widget API
</ParamField>

<ParamField body="company_id" type="integer" required>
  Identyfikator Twojej firmy
</ParamField>

<ParamField body="message" type="string" required>
  Tekst wiadomości użytkownika (maks. 2000 znaków)
</ParamField>

<ParamField body="session_id" type="string">
  Identyfikator sesji dla ciągłości rozmowy
</ParamField>

## Odpowiedź

Odpowiedź jest przesyłana strumieniowo jako Server-Sent Events (SSE):

```
data: {"chunk": "Hello"}
data: {"chunk": "! How"}
data: {"chunk": " can I"}
data: {"chunk": " help you?"}
data: [DONE]
```

## Przykład

```javascript
const response = await fetch('/api/widget/send-message', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    api_key: 'YOUR_API_KEY',
    company_id: 1,
    message: 'What products do you offer?',
    session_id: 'abc123'
  })
});

const reader = response.body.getReader();
const decoder = new TextDecoder();

while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  console.log(decoder.decode(value));
}
```
