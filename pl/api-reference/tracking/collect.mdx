---
title: Zbieranie zdarzeń
api: "POST /api/tracker/collect"
description: "Wysyłanie danych śledzenia (odsłony stron, zdarzenia)"
---

## Żądanie

<ParamField body="api_key" type="string" required>
  Twój klucz Widget API
</ParamField>

<ParamField body="company_id" type="integer" required>
  Identyfikator Twojej firmy
</ParamField>

<ParamField body="session_token" type="string" required>
  Unikalny identyfikator sesji
</ParamField>

<ParamField body="pageviews" type="array">
  Tablica obiektów odsłon stron (maks. 50)
</ParamField>

<ParamField body="events" type="array">
  Tablica obiektów zdarzeń (maks. 50)
</ParamField>

## Obiekt odsłony strony

```json
{
  "url": "https://example.com/products",
  "title": "Our Products",
  "referrer": "https://google.com",
  "timestamp": 1709640000
}
```

## Obiekt zdarzenia

```json
{
  "name": "form_submit",
  "category": "engagement",
  "data": {
    "form_id": "contact-form",
    "fields": ["name", "email", "phone"]
  },
  "timestamp": 1709640000
}
```

## Odpowiedź

```json
{
  "success": true
}
```

<Warning>
  Tablice są ograniczone do 50 elementów każda, aby zapobiec nadużyciom.
</Warning>
