---

## Request

<ParamField body="api_key" type="string" required>
  Your widget API key
</ParamField>

<ParamField body="company_id" type="integer" required>
  Your company ID
</ParamField>

<ParamField body="session_token" type="string" required>
  Unique session identifier (generated by tracker.js)
</ParamField>

<ParamField body="device" type="object">
  Device information
</ParamField>

<ParamField body="utm" type="object">
  UTM campaign parameters
</ParamField>

## Device Object

```json
{
  "type": "desktop",
  "browser": "Chrome",
  "os": "Windows 11",
  "screen": "1920x1080"
}
```

## UTM Object

```json
{
  "source": "google",
  "medium": "cpc",
  "campaign": "spring_sale",
  "term": "ai chatbot",
  "content": "banner_v2"
}
```

## Response

```json
{
  "success": true,
  "data": {
    "session_id": 12345
  }
}
```
