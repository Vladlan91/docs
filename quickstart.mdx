## 1. Create an Account

Sign up at [app.revol.com](https://app.revol.com/register) and create your first company.

## 2. Create an AI Agent

<Steps>
  <Step title="Go to AI Agents">
    Navigate to **Agents Platform** → **AI Agents** in the sidebar.
  </Step>
  <Step title="Create Agent">
    Click **Create Agent**, choose a use case, and configure basic settings.
  </Step>
  <Step title="Add Knowledge">
    Upload documents, add URLs, or write text content for your agent's knowledge base.
  </Step>
  <Step title="Configure Workflow">
    Set up the conversation workflow with nodes and conditions.
  </Step>
</Steps>

## 3. Install Widget

Add this script to your website before the closing `</body>` tag:

```html
<script src="https://app.revol.com/tracker.js"
        data-api-key="YOUR_API_KEY"
        data-company-id="YOUR_COMPANY_ID"
        async>
</script>
```

<Note>
  You can find your API key and Company ID in **Admin Panel** → **General Settings**.
</Note>

## 4. Test Your Agent

Open your website and interact with the chat widget. Your AI agent will respond based on its knowledge base and workflow configuration.
