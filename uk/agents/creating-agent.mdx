## Створення нового агента

1. Перейдіть до **Agents Platform** → **AI Agents**
2. Натисніть **Create Agent**
3. Заповніть основну інформацію:

| Поле | Опис |
|------|------|
| **Name** | Відображуване ім'я агента |
| **Company** | До якої компанії належить цей агент |
| **Use Case** | Продажі, підтримка, генерація лідів тощо |
| **LLM Provider** | OpenAI, Anthropic, Google або Groq |
| **Model** | Конкретна модель для використання |

## Налаштування System Prompt

System prompt — це основний набір інструкцій для вашого агента. Найкращі практики:

<Tip>
  Будьте конкретними щодо ролі, тону та меж агента. Додайте приклади того, як він повинен відповідати.
</Tip>

```text
You are a sales assistant for [Company Name].
Your goal is to help customers find the right product.

Rules:
- Always be polite and professional
- If you don't know the answer, say so honestly
- Never discuss competitor products
- Suggest relevant products from the knowledge base
```

## Додавання бази знань

Перейдіть на вкладку **Knowledge**, щоб додати контент:

- **Documents** — файли PDF, DOCX, TXT
- **URLs** — веб-сторінки для збору та індексації
- **Text** — текстовий контент напряму

<Note>
  Документи розбиваються на частини (chunks) та перетворюються на векторні embeddings для семантичного пошуку. Агент отримує найбільш релевантні частини під час кожної розмови.
</Note>

## Налаштування Workflow

Вкладка **Workflow** надає візуальний редактор:

1. Почніть з вузла **Start**
2. Додайте вузли **LLM** для відповідей AI
3. Використовуйте вузли **Condition** для розгалуження логіки
4. Додайте вузли **Tool Call** для зовнішніх дій
5. З'єднайте вузли ребрами для визначення потоку
