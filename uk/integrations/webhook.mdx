## Огляд

Custom Webhook — найпростіша інтеграція — вона надсилає HTTP-запити на будь-яку вказану вами URL-адресу. Використовуйте її для передачі даних лідів у CRM, запуску автоматизацій у Zapier/Make або підключення до будь-якої системи з API.

Без OAuth, без SDK — лише цільова URL-адреса та необов'язкові заголовки.

## Налаштування

<Steps>
  <Step title="Додайте інтеграцію">
    Перейдіть до **Інтеграції** → **Додати інтеграцію** → оберіть **Custom Webhook** у категорії CRM.
  </Step>
  <Step title="Налаштуйте">
    Введіть webhook URL та необов'язкові заголовки автентифікації (наприклад, `Authorization: Bearer ...`).
  </Step>
  <Step title="Перевірте">
    Revol надсилає тестовий payload для перевірки, що URL-адреса доступна та повертає відповідь 2xx.
  </Step>
</Steps>

## Інструменти агента

| Інструмент | Опис |
|---|---|
| `send_data` | Надіслати структуровані JSON-дані (інформація про ліда, дані форми) на webhook URL |
| `fetch_data` | Виконати GET-запит для отримання даних із зовнішнього API |
| `send_custom_request` | Надіслати довільний HTTP-запит з налаштовуваним методом, заголовками та тілом |

## Формат Payload

Коли агент викликає `send_data`, Revol надсилає POST-запит з такою структурою:

```json
{
  "event": "lead_captured",
  "timestamp": "2026-03-07T14:30:00Z",
  "company_id": 5,
  "agent_id": 12,
  "conversation_id": "conv_abc123",
  "data": {
    "name": "John Doe",
    "email": "john@example.com",
    "phone": "+380501234567",
    "message": "Interested in Enterprise plan",
    "source": "website_widget",
    "utm_campaign": "spring_promo"
  }
}
```

Вміст об'єкта `data` визначається AI-агентом на основі інформації, зібраної під час розмови.

## Сценарії використання

- **Інтеграція з CRM** — Передавайте кваліфікованих лідів у Pipedrive, HubSpot, Salesforce або будь-яку CRM
- **Zapier/Make** — Запускайте багатокрокові автоматизації на основі подій розмов
- **Slack/Discord** — Надсилайте сповіщення при захопленні цінних лідів
- **Власний бекенд** — Викликайте власний API для створення замовлень, оновлення записів або запуску workflow
