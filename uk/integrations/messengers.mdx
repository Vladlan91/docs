## Telegram

Telegram — найпростіша інтеграція — без OAuth, без залежностей від SDK. Лише токен бота від BotFather.

### Налаштування

<Steps>
  <Step title="Створіть бота">
    Відкрийте [@BotFather](https://t.me/BotFather) у Telegram, надішліть `/newbot` і дотримуйтесь інструкцій. Ви отримаєте **токен бота** на кшталт `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`.
  </Step>
  <Step title="Додайте інтеграцію">
    Перейдіть до **Інтеграції** → **Додати інтеграцію** → оберіть **Telegram** у категорії Месенджери.
  </Step>
  <Step title="Введіть токен">
    Оберіть компанію, дайте інтеграції назву та вставте токен бота.
  </Step>
  <Step title="Перевірте та збережіть">
    Revol викликає Telegram Bot API для перевірки токена та автоматично налаштовує webhook. Ваш бот починає отримувати повідомлення одразу.
  </Step>
</Steps>

### Інструменти агента

| Інструмент | Опис |
|---|---|
| `send_message` | Надіслати текстове повідомлення в чат |
| `send_file` | Надіслати документ, фото або медіафайл |
| `get_chat_history` | Отримати останні повідомлення з чату |
| `get_chat_info` | Отримати метадані чату/користувача |
| `edit_message` | Редагувати раніше надіслане повідомлення |
| `delete_message` | Видалити повідомлення |

### Потік Webhook

```
Користувач надсилає повідомлення в Telegram
    → Telegram пересилає на Revol webhook
    → Revol спрямовує до призначеного AI-агента
    → Агент обробляє та генерує відповідь
    → Revol викликає Telegram Bot API для надсилання відповіді
    → Користувач бачить відповідь у Telegram
```

## WhatsApp Business

<Note>
  Інтеграція з WhatsApp знаходиться в розробці (Phase 3). Цей розділ описує заплановану реалізацію.
</Note>

Доступ до WhatsApp Business API здійснюється через Meta Graph API. Потрібен акаунт Meta Business та налаштування WhatsApp Business API.

### Налаштування

<Steps>
  <Step title="Налаштування Meta Business">
    Створіть [акаунт Meta Business](https://business.facebook.com/) та налаштуйте доступ до WhatsApp Business API.
  </Step>
  <Step title="Додайте інтеграцію">
    Перейдіть до **Інтеграції** → **Додати інтеграцію** → оберіть **WhatsApp Business**.
  </Step>
  <Step title="OAuth">
    Натисніть "Підключити" — вас буде перенаправлено на екран згоди Meta. Авторизуйте доступ до вашого акаунту WhatsApp Business.
  </Step>
  <Step title="Перевірте та збережіть">
    Revol обмінює код на довготривалий токен (60 днів) та налаштовує webhook.
  </Step>
</Steps>

### Інструменти агента

| Інструмент | Опис |
|---|---|
| `send_message` | Надіслати текстове повідомлення |
| `send_template_message` | Надіслати попередньо затверджене шаблонне повідомлення |
| `send_media` | Надіслати зображення, документ або відео |
| `get_message_history` | Отримати історію розмови |
| `get_contact_info` | Отримати інформацію профілю контакту |
| `mark_as_read` | Позначити повідомлення як прочитані |

## Потік повідомлень (усі месенджери)

Незалежно від месенджера, потік обробки ідентичний:

<Steps>
  <Step title="Вхідне повідомлення">
    Платформа месенджера надсилає webhook до Revol із вмістом повідомлення та інформацією про відправника.
  </Step>
  <Step title="Маршрутизація до агента">
    Revol ідентифікує інтеграцію, знаходить призначеного AI-агента та створює або продовжує розмову.
  </Step>
  <Step title="AI-обробка">
    Повідомлення обробляється через workflow агента — LLM-вузли, пошук у базі знань RAG, виклики інструментів, умови.
  </Step>
  <Step title="Вихідна відповідь">
    Відповідь агента надсилається назад через API того ж месенджера. Користувач бачить відповідь у своєму чаті.
  </Step>
</Steps>
