## Огляд

Вкладка Sessions надає детальний перегляд усіх сесій відвідувачів. Переглядайте, фільтруйте та деталізуйте окремі подорожі відвідувачів, щоб зрозуміти, як люди взаємодіють з вашим сайтом.

## Таблиця сесій

Кожен рядок представляє одну сесію відвідувача:

| Колонка | Опис |
|---|---|
| **Guest Token** | Унікальний ідентифікатор відвідувача |
| **Country** | Країна відвідувача (іконка прапора) |
| **Device** | Desktop, Mobile або Tablet |
| **Browser** | Назва та версія браузера |
| **OS** | Операційна система |
| **Pages** | Кількість переглянутих сторінок |
| **Duration** | Тривалість сесії |
| **Source** | Джерело трафіку (UTM або referrer) |
| **Started** | Час початку сесії |

## Деталі сесії

Натисніть на будь-яку сесію, щоб відкрити детальний перегляд повної подорожі відвідувача:

- **Перегляди сторінок** — впорядкований список відвіданих сторінок з часовими мітками та часом на кожній сторінці
- **Події** — кастомні події, що спрацювали під час сесії
- **Глибина прокрутки** — наскільки далеко відвідувач прокрутив кожну сторінку
- **Розмова** — посилання на чат-розмову, якщо вона була розпочата

## Фільтри

Фільтруйте сесії за пристроєм, країною, джерелом, браузером, ОС, статусом відмови, кампанією та діапазоном дат — ті самі фільтри, що доступні на Dashboard.
