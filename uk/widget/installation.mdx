## Швидке встановлення

Додайте цей скрипт перед закриваючим тегом `</body>` на вашому сайті:

```html
<script src="https://app.revol.com/tracker.js"
        data-api-key="YOUR_API_KEY"
        data-company-id="YOUR_COMPANY_ID"
        async>
</script>
```

Цей скрипт забезпечує роботу **чат-віджета** та **відстеження відвідувачів** — додатковий код не потрібен.

## Конфігурація

| Атрибут | Обов'язковий | Опис |
|---------|--------------|------|
| `data-api-key` | Так | Ваш API ключ віджета |
| `data-company-id` | Так | Ваш ID компанії |

<Tip>
  Знайдіть API ключ та ID компанії в **Панелі адміністратора** → **Загальні налаштування**.
</Tip>

## Що завантажується

Скрипт трекера автоматично:

1. **Ініціалізує відстеження** — сесії, перегляди сторінок, події
2. **Завантажує віджет** — кнопку чату з вашим налаштованим зовнішнім виглядом
3. **Ліниво завантажує модулі** — тригери, оверлеї, підміна номерів (лише за потреби)

## Shadow DOM

Віджет рендериться всередині контейнера **Shadow DOM**, що означає:

- CSS вашого сайту не впливає на віджет
- CSS віджета не впливає на ваш сайт
- Повна ізоляція стилів

## Односторінкові додатки (SPA)

Трекер автоматично виявляє зміни URL через події `pushState` та `popstate`. Додаткова конфігурація для React, Vue або Angular додатків не потрібна.
