Вкладка Widget у панелі агента дозволяє повністю налаштувати, як чат-віджет виглядає та працює на вашому сайті. Всі зміни відображаються у превью в реальному часі.

## Кнопка Render

Кнопка **Render** у зоні превью віджета перезавантажує живий попередній перегляд з поточними налаштуваннями. Використовуйте її, щоб побачити, як виглядають ваші зміни перед збереженням.

## Appearance (Зовнішній вигляд)

Вкладка Appearance контролює візуальний дизайн віджета.

### Кольори

| Параметр | Опис |
|---|---|
| **Primary** | Основний колір кнопки та акцентів віджета |
| **Chat** | Колір кнопки дії чату |
| **Message** | Колір фону бульбашок повідомлень агента |
| **Animation** | Колір ефектів пульсації та свічення навколо кнопки віджета |
| **Rating** | Колір іконок зірок рейтингу після розмови |
| **Status Text** | Колір тексту статусу на кнопці віджета |

Натисніть на будь-який колірний зразок, щоб відкрити палітру з вибором насиченості/відтінку та полем для hex-коду.

### Текстові мітки

| Параметр | За замовчуванням | Опис |
|---|---|---|
| **Main Title** | "Need help?" | Заголовок, що відображається на кнопці віджета |
| **Call Button** | "Web call" | Підпис на кнопці голосового дзвінка |
| **Chat Button** | "Chat" | Підпис на кнопці чату |
| **Input Placeholder** | "Enter the text" | Текст-підказка у полі введення повідомлення |

### Позиція

| Варіант | Опис |
|---|---|
| **Bottom Right** | За замовчуванням — віджет з'являється у правому нижньому куті |
| **Bottom Center** | По центру внизу сторінки |
| **Bottom Left** | Лівий нижній кут |
| **Top Right** | Правий верхній кут |
| **Top Center** | По центру вгорі |
| **Top Left** | Лівий верхній кут |

### Розмір та стиль

| Параметр | Варіанти | Опис |
|---|---|---|
| **Widget Theme** | Light / Dark | Колірна тема вікна чату |
| **Widget Size** | Small / Medium / Large | Загальний розмір віджета |
| **Border Radius** | 0–50px (слайдер) | Заокруглення кутів віджета |
| **Background Opacity** | 0–100% (слайдер) | Прозорість фону віджета |
| **Avatar URL** | URL | Кастомне зображення аватара агента в чаті |

## Behavior (Поведінка)

Вкладка Behavior контролює, як віджет функціонує.

### Мова

| Параметр | Варіанти | Опис |
|---|---|---|
| **Widget Language** | English, Українська, Polski, Deutsch, Español | Мова інтерфейсу для всіх міток та системних повідомлень віджета |

### Функції

| Параметр | За замовчуванням | Опис |
|---|---|---|
| **Feedback Collection** | Увімк. | Показувати зірковий рейтинг після завершення розмови |
| **Text During Call** | Увімк. | Дозволити користувачу набирати текстові повідомлення під час голосового дзвінка |
| **Chat Only Mode** | Вимк. | Повністю вимкнути голосові дзвінки — доступний лише текстовий чат |
| **Terms & Conditions** | Вимк. | Вимагати прийняття умов перед початком чату |
| **Sound Effects** | Увімк. | Звукові сповіщення при нових повідомленнях |

### Автоматизація

| Параметр | За замовчуванням | Опис |
|---|---|---|
| **Auto Open** | Вимк. | Автоматично відкривати віджет після затримки |
| **Auto Open Delay** | 5с (слайдер, 0–60с) | Час очікування перед автовідкриттям (лише коли Auto Open увімкнено) |
| **Auto Greeting** | Вимк. | Автоматично надсилати привітальне повідомлення при відкритті віджета |

### Видимість

| Параметр | За замовчуванням | Опис |
|---|---|---|
| **Widget Enabled** | Увімк. | Головний перемикач — показувати чи приховувати віджет на вашому сайті |
| **Show on Mobile** | Увімк. | Відображати віджет на мобільних пристроях |
| **Show on Desktop** | Увімк. | Відображати віджет на десктопних пристроях |

### Брендинг

| Параметр | За замовчуванням | Опис |
|---|---|---|
| **Powered By Badge** | Увімк. | Показувати напис "Powered by Revol" внизу віджета |
| **Welcome Message** | — | Привітальне повідомлення, що відображається при відкритті чату (до першого повідомлення користувача) |

### Деталі Terms & Conditions

Коли **Terms & Conditions** увімкнено, з'являються додаткові поля:

| Параметр | Опис |
|---|---|
| **Terms Text** | Текст, що показується користувачу (напр., "Продовжуючи, ви погоджуєтесь з Умовами використання") |
| **Terms URL** | Посилання на повну сторінку умов |

## Voice (Голос)

Вкладка Voice налаштовує розпізнавання мовлення та синтез голосу для голосових дзвінків.

### Speech Recognition (STT)

| Параметр | Варіанти | Опис |
|---|---|---|
| **Language** | Українська, English | Мова для розпізнавання мовлення |

### Greeting Message (Привітання)

Текст, який агент промовляє при відповіді на голосовий дзвінок. Ви можете **Preview** (прослухати) аудіо перед збереженням та **Save** для застосування.

### Farewell Message (Прощання)

Текст, який промовляється при завершенні дзвінка. Також має кнопки **Preview** та **Save**.

### Voice Output (TTS)

| Параметр | Варіанти | Опис |
|---|---|---|
| **Provider** | OpenAI TTS | Провайдер синтезу мовлення (ElevenLabs та Google Cloud TTS скоро) |
| **Voice** | Alloy, Echo, Fable, Onyx, Nova, Shimmer | Характер голосу агента |
| **Model** | tts-1, tts-1-hd | Якість аудіо — стандартна або HD |
| **Speed** | 0.5x – 2.0x (слайдер) | Швидкість мовлення |

**Опис голосів:**

| Голос | Характер |
|---|---|
| **Alloy** | Нейтральний, збалансований |
| **Echo** | Теплий, розмовний |
| **Fable** | Виразний, британський акцент |
| **Onyx** | Глибокий, авторитетний |
| **Nova** | Дружній, оптимістичний |
| **Shimmer** | Чистий, м'який |

### Test Voice (Тест голосу)

Введіть будь-який текст та натисніть **Play Test**, щоб прослухати, як обраний голос звучить з поточними налаштуваннями.

## Save та Reset

- **Save** — зберегти всі зміни віджета
- **Reset** — повернути всі налаштування до останнього збереженого стану
