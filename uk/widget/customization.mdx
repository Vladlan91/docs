## Візуальні налаштування

Налаштуйте ваш віджет у **Панелі адміністратора** → **Загальні налаштування** → **Налаштування віджета**.

### Кольори

| Параметр | Опис |
|----------|------|
| **Колір кнопки** | Колір основної кнопки дії |
| **Колір анімації** | Ефекти пульсації та свічення |
| **Колір фону** | Фон вікна чату |
| **Фон повідомлення** | Колір бульбашки повідомлення асистента |

### Розташування

| Параметр | Опис |
|----------|------|
| **Радіус заокруглення** | Заокруглення кутів віджета (0-20px) |
| **Позиція** | Внизу ліворуч або внизу праворуч |
| **Мова віджета** | Мова інтерфейсу (EN, UK, PL, DE, ES) |

### Брендинг

| Параметр | Опис |
|----------|------|
| **Аватар** | Зображення аватара агента |
| **Привітальне повідомлення** | Текст початкового привітання |
| **Брендинг** | Значок "Powered by Revol" (можна прибрати на Premium+) |

## Темна тема

Віджет автоматично визначає системні налаштування користувача та перемикається між світлою та темною темами.
