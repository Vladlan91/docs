## Огляд

Вкладка Activity Log показує стрічку останніх 500 подій відвідувачів у реальному часі. Використовуйте для моніторингу живої активності, налагодження відстеження подій або перевірки правильності спрацювання тригерів та подій.

## Записи журналу

Кожен запис у журналі активності показує:

| Колонка | Опис |
|---|---|
| **Time** | Часова мітка події |
| **Type** | Тип події: Pageview, Event, Trigger, Conversation, Phone Swap |
| **Details** | Деталі конкретної події (URL сторінки, назва події, назва тригера тощо) |
| **Session** | Посилання на пов'язану сесію |
| **Device** | Тип пристрою відвідувача |
| **Country** | Країна відвідувача |

## Фільтри

Фільтруйте журнал активності за типом подій:

- **All** — показати всі події
- **Pageviews** — лише завантаження сторінок
- **Events** — лише кастомні події
- **Triggers** — лише активації тригерів
- **Conversations** — розпочаті чат-розмови
- **Phone Swaps** — підміни номерів телефону

## Використання

Журнал активності корисний для:

- **Налагодження** — перевірте, що події та тригери спрацьовують як очікується
- **Живий моніторинг** — спостерігайте за активністю відвідувачів у реальному часі
- **Швидкі перевірки** — перегляньте нещодавню активність без навігації через сесії
