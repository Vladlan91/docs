## Огляд

Вкладка Triggers дозволяє створювати автоматичні дії, які спрацьовують на основі правил поведінки відвідувачів. Тригери можуть показувати попапи, оверлеї, панелі сповіщень або кастомний HTML при виконанні певних умов.<Frame>

## Створення тригера

Натисніть **Create Trigger**, щоб налаштувати нову автоматичну дію:

| Поле | Опис |
|---|---|
| **Name** | Назва тригера для ідентифікації |
| **Type** | Тип відображення: Popup, Overlay, Notification Bar або Custom HTML |
| **Content** | Повідомлення або HTML-контент для відображення |

## Умови тригера

Визначте, коли тригер має спрацювати, використовуючи одну або кілька умов:

| Умова | Опис |
|---|---|
| **Time on Page** | Спрацювати після X секунд перебування на сторінці |
| **Scroll Depth** | Спрацювати при прокрутці за визначений поріг відсотків |
| **Exit Intent** | Спрацювати коли відвідувач рухає мишу до закриття/назад браузера |
| **Page Count** | Спрацювати після перегляду X сторінок у сесії |
| **Page URL** | Спрацювати лише на конкретних сторінках |
| **Device** | Спрацювати лише на Desktop, Mobile або Tablet |
| **New/Returning** | Спрацювати лише для нових або повторних відвідувачів |

## Налаштування тригера

| Параметр | Опис |
|---|---|
| **Frequency** | Частота показу: Раз за сесію, Раз за відвідувача або Кожного разу |
| **Delay** | Додаткова затримка в секундах перед відображенням |
| **Enabled** | Перемикач активації/деактивації тригера |

## Список тригерів

Таблиця тригерів показує всі визначені тригери з:

| Колонка | Опис |
|---|---|
| **Name** | Назва тригера |
| **Type** | Тип відображення |
| **Conditions** | Підсумок умов тригера |
| **Fires** | Загальна кількість активацій |
| **Enabled** | Статус активності |
| **Actions** | Редагувати або видалити тригер |
