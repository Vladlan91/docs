## Огляд

Вкладка Campaigns дозволяє створювати та керувати кампаніями відстеження з UTM-параметрами. Кожна кампанія генерує унікальну URL-адресу відстеження, яка атрибутує сесії відвідувачів до конкретних маркетингових зусиль.<Frame>

## Створення кампанії

Натисніть **Create Campaign**, щоб налаштувати нову кампанію:

| Поле | Опис |
|---|---|
| **Name** | Назва кампанії для внутрішнього використання |
| **Source** | Джерело трафіку (напр., google, facebook, newsletter) |
| **Medium** | Маркетинговий канал (напр., cpc, email, social, banner) |
| **Content** | Ідентифікатор контенту або варіанту оголошення (необов'язково) |
| **Term** | Ключове слово платної пошуки (необов'язково) |

Після створення система генерує URL відстеження з відповідними UTM-параметрами, який ви можете використовувати у маркетингових каналах.

## Список кампаній

Таблиця кампаній показує всі ваші кампанії з ключовими метриками:

| Колонка | Опис |
|---|---|
| **Name** | Назва кампанії |
| **Source** | Значення UTM source |
| **Medium** | Значення UTM medium |
| **Sessions** | Загальна кількість сесій, атрибутованих цій кампанії |
| **Created** | Дата створення кампанії |
| **Actions** | Редагувати або видалити кампанію |

## UTM-параметри

Кампанії використовують стандартні UTM-параметри, що додаються до URL вашого сайту:

- `utm_source` — ідентифікує джерело трафіку
- `utm_medium` — ідентифікує маркетинговий канал
- `utm_campaign` — ідентифікує назву кампанії
- `utm_content` — розрізняє варіанти оголошень (необов'язково)
- `utm_term` — ідентифікує ключові слова платної пошуки (необов'язково)

Коли відвідувач переходить за URL з UTM-параметрами, трекер автоматично атрибутує сесію відповідній кампанії.
