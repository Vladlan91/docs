## Request

<ParamField body="api_key" type="string" required>
  Ваш API key віджета
</ParamField>

<ParamField body="company_id" type="integer" required>
  ID вашої компанії
</ParamField>

<ParamField body="message" type="string" required>
  Текст повідомлення користувача (максимум 2000 символів)
</ParamField>

<ParamField body="session_id" type="string">
  Ідентифікатор сесії для підтримки безперервності розмови
</ParamField>

## Response

Відповідь передається потоково як Server-Sent Events (SSE):

```
data: {"chunk": "Hello"}
data: {"chunk": "! How"}
data: {"chunk": " can I"}
data: {"chunk": " help you?"}
data: [DONE]
```

## Приклад

```javascript
const response = await fetch('/api/widget/send-message', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    api_key: 'YOUR_API_KEY',
    company_id: 1,
    message: 'What products do you offer?',
    session_id: 'abc123'
  })
});

const reader = response.body.getReader();
const decoder = new TextDecoder();

while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  console.log(decoder.decode(value));
}
```
